# CareerBridge Platform

CareerBridge is a web-based platform that connects students with internship and attachment opportunities while helping employers find qualified candidates.

## Problem
Many students struggle to find internships and attachments, while companies find it difficult to reach suitable candidates.

## Solution
CareerBridge provides a centralized platform where:
- Students can create profiles and apply for opportunities.
- Employers can post opportunities and manage applications.
- Administrators can oversee platform activities.

## Planned Features
- Student registration and login
- Employer registration and login
- Internship posting
- Application tracking
- CV upload
- Admin dashboard
- Email notifications
